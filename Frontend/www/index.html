Правила сервісу. Для всіх-всіх-всіх

1. Тільки зареєстровані користувачі мають право брати участь у книгооберті.
2.Для реєстрації необхідно додати до профайлу не менше трьох книг.
3.Про срок видачі книжок користувачі домовляються особисто.
4.Користувач має сам оновлювати статус своїх книжок.
5.Пошкодження книжки може стати підставою для заборони видачі інших книг користувачу.
6.Адміністрація не несе відповідальність за особистий книгооберт користувачів, а тільки створює платформу для проведення такої взаємодії.
